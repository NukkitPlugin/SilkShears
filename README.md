# SilkShears
When you break a crop with "silk touch" enchanted shears,
			It returns to when the crop was first planted.


## How to use in maven (pom.xml)
Add the following source to pom.xml:
```
<repositories>
   <repository>
      <id>organization</id>
      <url>https://github.com/nukkitplugin/maven-repo/raw/master/</url>
   </repository>
</repositories>
<dependencies>
   <dependency>
      <groupId>nukkitplugin</groupId>
      <artifactId>SilkShears</artifactId>
      <version>1.0.0-SNAPSHOT</version>
      <scope>provided</scope>
   </dependency>
</dependencies>
```
